# Cherry-Blossom-Fastfetch
Custom Kitty Terminal and Fastfetch config files for a Cherry Blossom Theme

## Download
Run
```
git clone https://github.com/Neptune3013/Cherry-Blossom-Kitty-Fastfetch.git
```
## Installation

### Kitty Terminal theme config

Copy Cherry_Blossom.conf into ~/.config/kitty/ <br>
(if the directory does not exist, open kitty terminal and presee ctrl+shift+F2, it will create the dir and the kitty.conf file)

add ``` include ./Cherry_Blossom.conf ``` to kitty.conf in ~/.config/kitty/ 

### Fastfetch config

copy config.jsonc to ~/.config/fastfetch/
